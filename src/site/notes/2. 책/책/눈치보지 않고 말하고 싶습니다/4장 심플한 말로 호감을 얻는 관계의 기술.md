# 재밌게 말하고 싶다면 BCD를 기억하세요
호모 나랜스 : 이야기꾼 인간
- B(Backgroud, 배경) : 이야기의 시간적, 공간적 배경을 말한다
	언제 어디서 일어난 일인지 밝힘으로써 전체적인 배경 그림을 그린다
- C(context, 맥락) : 그 사건에 대한 맥락을 설명하는 것
	본격적인 이야기를 풀기 전, 청중이 알아 놓으면 좋을 정보나 이해에 꼭 필요한 사하을 부연하여 전달하는 것
- D(drama, 서사) : 이야기를 잘 풀어낼 것
	시간이 흘러간대로 순차적으로 이야기를 풀 것
	문장은 짧은 단문 형태로 서술
	각 문장을 접속사를 잘 활용해서 이어가면 흥미 유발에도 도움이 되고 말하기가 편해진다
	감각어와 감정표현을 충분히 써주면 청중의 몰입도에 도움이 된다

# 수용하는 태도는 언제나 괜찮아요
> 동조 화법

유사성의 원리 : 흔히 본인가 다른 사람에게 끌린다고 생각하지만, 심리학 실험 결과들을 보면 사람들은 자신과 유사한 사람에게 끌린다고 한다

- 먼저 받아들이는 ==수용적 태도== 생각하기
- 일단은 열린 마음으로 들어주기
- 눈썹을 살짝 올린다는 느낌으로 표정을 열고 고개를 끄덕이면서 부드럽게 “음~”, “어~”와 같은 소리 내보기
- 재진술
	반복 : “다음 달에 퇴사한다고?”
	환언 : ”그러니까 번아웃이 왔다는 거구나“
	요약 : “엄마랑 크게 다퉜구나”
- 비슷한 점 어필하기

## 공감의 속성 4가지
1. 타인의 관점으로 세상을 본다
2. 비판하지 않는 태도를 갖는다
3. 타인의 감정을 이해한다
4. 타인이 감정을 이해한다는 사실을 표현한다
