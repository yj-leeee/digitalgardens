# 문장에서 가장 중요한 처음과 끝
- 두괄식 말하기 : 내가 말하고자 하는 핵심을 가장 먼저 내세우는 것
	별 내용이 아니어도 **또렷한 인상**을 줄 수 있다는 장점
- 확실한 끝 처리

최신효과 : 가장 **나중에 들어온 정보 값**이 존재감을 많이 발휘함

미안하다는 말 대신 **구체적으로 감사해 버릇하기**

멋지게 발표를 하기 위해서는 먼저 **글을 잘 읽는 훈련**하기

## 이야기하듯 말하는 방법
1. 원고 고치기
2. 편하게 쓰인 원고를 바탕으로 본격적으로 리딩훈련하기
3. 감정 넣기
