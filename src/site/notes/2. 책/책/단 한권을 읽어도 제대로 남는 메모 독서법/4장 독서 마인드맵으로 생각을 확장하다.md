## 책 내용을 요약하기 위해 사용하는 방법 세가지

1. 키워드 뽑기(주가지 위에 들어가는 항목)
2. 범주화-계층형 목록만들기 : 일정한 기준으로 정보를 묶는 작업
3.   강조 처리 : 정보의 중요도에 따라 가지의 색상이나 기호등을 사용하여 강조 처리 하기
> 독서 마인드 맵으로 책 내용을 전달하고, 글쓰기 등에 활용할 수 있으면 잘 만든 것

  

독서 노트 = 나무 한 그루 관찰하기

독서 마인드맵 = 숲 전체 바라보기
